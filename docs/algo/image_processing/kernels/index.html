Spatial Filters

  • To keep the same size of output after filtering.
    • Zero padding
    • Replicate padding

Average Filters​

  • Characteristic: Low-pass filters
    • Removes high-frequency components
  • Any filters that sums to 1, eg
    • [0001/31/31/3000]\begin{bmatrix}0&0&0\\1/3&1/3&1/3\\0&0&0\end{bmatrix}​01/30​01/30​01/30​​

Box Filter​

  • Box filter of size nnn.
  • Problems:
    • Non-isotropic (orientation-dependent)
      • Creates unnatural, stepped patterns in the output image.
    • Sharp boundary

Gaussian Filter​

  • k(x,y)=A⋅exp⁡(−x2+y22σ2)k(x,y)=A\cdot\exp{\left(-\frac{x^2+y^2}{2\sigma^2}\right)}k(x,y)=A⋅exp(−2σ2x2+y2​)
  • AAA chosen to make the kernel sums to 1.
  • Normally truncated at around 2σ2\sigma2σ radius.

Sharpening Filters​

  • Increase high-frequency components
    • Non-trivial
  • Image sharpening can be performed by inverting the blurring process:
    • Isharpen=I+α(1−Fblur(I))I_\text{sharpen}=I+\alpha(1-F_\text{blur}(I))Isharpen​=I+α(1−Fblur​(I))
  • Hence, sharpening kernel can be given by:
    • Fsharpen=((1+α)Fidentity−αFblur)F_\text{sharpen}=((1+\alpha)F_\text{identity}-\alpha F_\text{blur})Fsharpen​=((1+α)Fidentity​−αFblur​)
    • Pre-computing this kernel allow one-pass application over the image (more efficient)
  • Problem:
    • Introduces halo artifacts.
  • Median Filter:
    • No halo artifacts
