Multi-Scale Structural Similarity Measure (MS-SSIM)​

SSIM​

SSIM(x,y)=[l(x,y)]α⋅[c(x,y)]β⋅[s(x,y)]γ\begin{align*} SSIM(x,y)=[l(x,y)]^\alpha\cdot [c(x,y)]^\beta\cdot [s(x,y)]^\gamma \end{align*}SSIM(x,y)=[l(x,y)]α⋅[c(x,y)]β⋅[s(x,y)]γ​

with

  • Mean brightness estimation l(x,y)=2μxμy+C1μx2+μy2+C1l(x,y)=\frac{2\mu_x\mu_y+C_1}{\mu_x^2+\mu_y^2+C_1}l(x,y)=μx2​+μy2​+C1​2μx​μy​+C1​​
    • Using mean to estimate the mean brightness
  • Mean contrast estimation c(x,y)=2σxσy+C2σx2+σy2+C2c(x,y)=\frac{2\sigma_x\sigma_y+C_2}{\sigma_x^2+\sigma_y^2+C_2}c(x,y)=σx2​+σy2​+C2​2σx​σy​+C2​​
    • Using the variance to estimate the contrast
  • Structural Similarity s(x,y)=σxy+C3σxσy+C3s(x,y)=\frac{\sigma_{xy}+C_3}{\sigma_x\sigma_y+C_3}s(x,y)=σx​σy​+C3​σxy​+C3​​
    • Using the covariance to estimate the structural similarity.
  • where, C1=(K1L)2,C2=(K2L)2,C3=C2/2C_1=(K_1 L)^2, C_2=(K_2 L)^2, C_3=C_2/2C1​=(K1​L)2,C2​=(K2​L)2,C3​=C2​/2

    • with default α=β=γ=1\alpha=\beta=\gamma=1α=β=γ=1.

  • Symmetric: SSIM(x,y)=SSIM(y,x)SSIM(x,y)=SSIM(y,x)SSIM(x,y)=SSIM(y,x)
