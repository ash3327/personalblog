Image Storing and File Sizes​

Images​

Consider an image of size W×HW\times HW×H with CCC color channels, with LLL bits storing the scale per channel, the uncompressed file size is:

W×H×C×(L/8)W\times H\times C\times (L/8)W×H×C×(L/8)

Video​

Consider a video of size W×HW\times HW×H with CCC channels, 8 bit per pixel, duration of LLL seconds, and a frame-rate of FFF fps, then the uncompressed file size is:

W×H×C×L×FW\times H\times C\times L\times FW×H×C×L×F
