Approaches​

  • Discriminative Model
    • P(Y∣X=x)P(Y|X=x)P(Y∣X=x) models the conditional probability of target YYY, given an observation xxx.
  • Generative Model
    • Models the joint probability distribution P(X,Y)P(X,Y)P(X,Y) given observable variable XXX and target variable YYY.
    • Auto-Encoder:
    • Diffusion
    • GAN
      • Problems:
        • Mode collapse
        • Non-convergence (oscillation in parameters)
        • Diminished gradients
        • Overfitting (unbalance between generator and discriminator)
      • Later Works:
        • StyleGAN, BigGAN: Conditioning
      • GAN inversion: Find E(I)=zE(I)=zE(I)=z for G(z)=IG(z)=IG(z)=I.

  • Solution:
